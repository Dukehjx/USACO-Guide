# Self-Test Problem: Greet Your Friend

**Statement:**  
Write a program that reads a name from a the user input, and writes `Hello, <name>!` in ther terminal.

---

**Input Format:**  
A single word: the name.

**Output Format:**  
A single line: `Hello, <name>!`

---

**Sample Input:**
```cpp
Duke
```
**Sample Output:**
```cpp
Hello, Duke!
```
