# Self-Test Problem: Count Unique Numbers

**Statement:**  
Given N integers, print the number of unique values.

---

**Input Format:**  
First line: integer N (1 ≤ N ≤ 100)  
Second line: N integers

**Output Format:**  
A single integer: the number of unique numbers

---

**Sample Input:**
```
7
1 2 3 2 3 4 1
```

**Sample Output:**
```
4
```
